### 🔄 Power BI Report Migration: Environment & Database Transition
> **Power BI · DAX · SQL Server · MySQL**

Seamlessly migrated a production BI report across both environments and database platforms with **zero data loss**.

- 🔍 Profiled inventory data across test (99 records) vs production (1,043 records); resolved 2 mismatched IDs via SQL UPDATE
- 📐 Built **6 DAX measures** (Supply Shortage, Profit/Loss, Avg Daily Loss) using `DIVIDE`, `SUMX`, `FILTER`
- ✅ Validated all 6 KPIs post-migration — **100% match** across SQL Server → MySQL and test → production environments
